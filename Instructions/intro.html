Summary: Latitude vs. X

The purpose of this project was to analyze weather changes depending on a latitude distance from the equator. Data for the random 500 cities was collected utilizing OpenWeatherMap API, cleaned, and assembled as a data frame.

After assembling the data frame, various aspects of the weather vs. latitude were plotted on the individual plots using Matplotlib and analyzed for a correlation between the latitude distance from the equator on both hemispheres and the weather conditions. Analysis was performed considering the following factors: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as a part of the analysis, as well as explanations and descriptions of any trends and correlations which were observed during the study of the produced plots.
